Features

Eliminating Geographical Barriers

A researcher desiring to carry out an experiment in Los Angeles can simply have the participants there download their customized tracking app. Larger scale studies can be conducted on specific regions using the data from participants who agree to use location services.

Push Notifications

Never miss a data point again, as the platform allows researchers to send push notifications at any time as a reminder to collect that crucial data point. Moreover, scheduled push notifications can be set up by both the participant and the researcher to ensure proper data collection

Optimization

Instead of spending months gathering and collecting data, create studies on the platform and enable participants to share their preexisting and/or future data with you.

Individual Web Page

Each researcher can create a customized web page using simple short codes the framework provides. This not only enables data collection to occur in one simple standardized place, but also allows easy researcher-to-participant communication

Large Data Sets

Just 1 active participant per 1,000 individuals in Los Angeles would result in 10,000 active participants constantly tracking and providing health data (both physical and emotional) waiting to be analyzed. Read the policy and promise about the data you receive!

Symbiotic Relationship

The number of consistent data means the capability for truly ground breaking statistical research. People are depending on technology like never before. It is time to give researchers an opportunity to put that data to uses that are mutually beneficial to both parties.
